# TypeDown

A minimal, elegant markdown editor built with Tauri, SvelteKit, and CodeMirror.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://rustup.rs/)
- [Tauri prerequisites](https://tauri.app/start/prerequisites/)

## Getting Started

```bash
git clone https://github.com/gupsammy/typedown.git
cd typedown
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

Binaries will be in `src-tauri/target/release/`.
